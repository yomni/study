# Spring Batch 스터디

## 시나리오
1. CSV 파일에서 커피 목록을 가져오고 
2. 사용자 정의 프로세서를 사용하여 변환하고 
3. 메모리 내 데이터베이스에 최종 결과를 저장하는 작업

## 용어정리
Job : 작업 목표 전체  
    - Flow 를 어떻게 정의 할 지 Job 에서 결정, 이 때 단일 Step 단위로도 Flow 단위로도 지정 가능
Flow : Step 을 단일 / 여러개를 묶어놓은 것
Step : 각 단계에서 해야되는 일 정의
